Tamanhos de imagens

Mova o navegador para ver os tamanhos de imagens sendo alterados.

Para esse recurso usamos a tag picture e dentro dela usamos a tag img.

Não se esquercer de colocar o tipo de unidade no tamanho escolhido. Ex: pixel.

Adicionando áudios ao site.

Para a inscersão de áudios no HTML5 podemos usar apenas a tag audio e inserir o arquivo. Porém também também podemos aumentar a tag e usar no formato

audio preload="metadata" autoplay controls loop
