Trabalhando com listas

Listas ordenadas

Para a criação de listas, usamos a tag ol, já para a alteração do tipo de lista, dentro da tag ol, colocamos a tag type.

  1. Acordar
  2. Tomar café
  3. Ligar para o João
  4. Escovar os dentes

Listão não ordenadas

Usado para criação de listas que não são necessariamente sequenciais. Para isso, usamos a tag ul, e usamos a tag type, para o tipo de lista.

  • Pão
  • Leite
  • Alface

Lista de definição

Lista usada para definir algum texto. As tags usadas são:

dl, que significa: definition list.

dt, que significa: definition term

dd, que significa: definition descrição

Exemplo:

JavaScript
Linguaguem de programação visando interatividade com o site.
